# NHK Plus Board

NHK+のお気に入り番組を設定することで次のようなことができます。
モバイル版での表示にも対応しています。

- 配信中の番組をWeb画面で一覧表示
- 直近に配信された番組や、まもなく配信終了する番組をLINEやDiscordに通知
